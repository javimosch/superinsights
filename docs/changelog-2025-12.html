Changelog

December 2025

📋

Reports engine

Full reporting system with generation, filtering templates, and viewing capabilities. Create and manage analytics reports directly from the UI.

🧩

Vue 3 UI components

New Vue 3-based skeleton loader, advanced modal/popover systems, and responsive navbar with mobile drawer. Breadcrumbs across admin pages for easier navigation.

📖

Documentation & GEMINI workflows

Comprehensive docs for reports, AI Analysis, external app integration, and internal services. GEMINI workflow documentation for AI-assisted development flows. SDK examples with cURL.
